* This game is played with 2 players. Whichever player has the most points by grabbing the glass objects at the end of the 2 minute timer countdown wins. 
* Player 1 uses W(up), A(left), S(down), D(right). Player 2 uses up arrow key to go up, down arrow key to go down, left arrow key to go left, and right arrow key to move right. 
* To jump: For player 1, spacebar must be pressed to jump. For player 2, right shift must be pressed to jump. 
* If either player hits a wall, a point gets deducted. Also, when a player ball jumps and collides onto the other player at the lower altitude, the person on the higher altitude gets added 1 point.
